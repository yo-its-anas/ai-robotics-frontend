# Part VI: Hardware Lab

## Part Overview

Physical AI requires proper hardware infrastructure: simulation workstations, edge devices, robot platforms, and cloud resources.

## Chapters
- **Ch 20**: High-performance simulation workstations (RTX GPUs)
- **Ch 21**: Jetson edge AI kits, ROS deployment
- **Ch 22**: Robot platform options (quadrupeds, humanoids)
- **Ch 23**: Cloud infrastructure (AWS, Omniverse Cloud)

## Learning Outcomes
1. Configure optimal development workstation
2. Deploy ROS 2 to Jetson edge devices
3. Evaluate robot platform trade-offs
4. Set up cloud-based workflows
